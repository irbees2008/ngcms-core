# Обновление: Опциональная мультиязычность

## Что добавлено

Теперь можно **включать/выключать систему переводов** для каждого мультисайта отдельно!

## 🎯 Зачем это нужно?

Не на всех сайтах нужны переводы статей. Например:

- **ru.web.ua** - основной сайт с переводами → мультиязычность **включена** ✅
- **blog.web.ua** - блог без переводов → мультиязычность **отключена** ❌
- **en.web.ua** - английская версия с переводами → мультиязычность **включена** ✅

## ✨ Как использовать

### При добавлении нового мультисайта:

1. **Админ панель → Конфигурация → Управление мультисайтами**
2. **Добавить новый сайт**
3. Заполните поля:
   - ID сайта: `blog_site`
   - Домены: `blog.web.ua`
   - Язык сайта: `Русский (ru)`
   - **☑ Мультиязычность** ← Новый чекбокс!
     - **Отмечено** (по умолчанию) - переводы **включены**
     - **Не отмечено** - переводы **отключены**

### Для существующих сайтов:

Отредактируйте `conf/multiconfig.php` вручную:

```php
$multiconfig = [
    'main' => [
        'domains' => ['ru.web.ua'],
        'active' => 1,
        'lang' => 'ru',
        'multilang_enabled' => 1,  // Переводы включены
        'db' => [...]
    ],
    'blog_site' => [
        'domains' => ['blog.web.ua'],
        'active' => 1,
        'lang' => 'ru',
        'multilang_enabled' => 0,  // Переводы ОТКЛЮЧЕНЫ
        'db' => [...]
    ]
];
```

## 📝 Что происходит при отключении

Когда `multilang_enabled = 0`:

- ❌ Блок "Переводы" **не отображается** в редакторе новостей
- ❌ Сайт **не участвует** в списке доступных языков для переводов
- ✅ Статьи сохраняются **как обычно** (все работает без изменений)
- ✅ Можно **в любой момент включить** обратно

## 🔧 Измененные файлы

1. **engine/skins/default/tpl/multisite.tpl** - чекбокс "Мультиязычность"
2. **engine/actions/configuration.php** - сохранение флага `multilang_enabled`
3. **engine/actions/news.php** - проверка флага перед показом переводов
4. **engine/includes/multilang.php** - фильтрация сайтов с отключенными переводами
5. **engine/lang/russian/admin/configuration.ini** - языковые строки
6. **engine/lang/english/admin/configuration.ini** - языковые строки

## ⚡ Обратная совместимость

Для существующих мультисайтов **без явно указанного** `multilang_enabled`:

- По умолчанию переводы **ВКЛЮЧЕНЫ** (для совместимости)
- Блок переводов будет работать как раньше
- Можно явно отключить, добавив `'multilang_enabled' => 0`
